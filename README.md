phpkg-rest
==========

インストール
----------
composerからこちらを叩く事でpokelaboのRESTフレームワークがインストールされます。

    {
        "require": {
            "pokelabo/rest": "dev-master"
        },
        "repositories": [
            {
                "type": "composer",
                "url": "http://composer.platform-test.pokelabo.jp/"
            }
        ],
        "minimum-stability": "dev"
    }

※exsampleがついてますので、構築の参考にしてください。
